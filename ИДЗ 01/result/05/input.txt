        <h1>Numeric/Text<a name="kanchor220"></a></h1>
        <p>Use the Numeric/Text format if the data entered into a field needs to be formatted as numbers <i>or</i> text.</p>
        <p>For example, you may have an <i>Additional</i> coverage field such as <i>Glass Breakage</i>. For some policies, this would display a monetary numerical amount on the form, such as $100. However, other policies might require that this be a text value, such as <i>Included</i> or <i>Not Applicable</i>. The Numeric/Text field type allows either type of data to be entered into the field.</p>
        <p>If the field value is numeric, then the numeric format will be applied, otherwise the text will be unchanged.</p>
        <p>For example, with a format of $ZZ9.99, the value 10 would appear as $10.00, but a text value of N/A would appear unchanged by the format.</p>
